# Как запустить проект ?

Вставить в командную строку 

```
git clone https://github.com/slowin1/web-project.git
```

После загрузки проекта 

Открыть терминал в VsCode или любой другой текстовый редактор далее переходим в корень проекта 

```
npm install --save three
npm install
npm run dev
```
